    Configuration type.

    Parameters:
    :   * **follow\_height\_m** (*float*) – [m] Follow height in meters (recommended minimum 8 meters)
        * **follow\_distance\_m** (*float*) – [m] Follow distance to target in meters (recommended minimum 4 meter)
        * **responsiveness** (*float*) – How responsive the vehicle is to the motion of the target, Lower value = More responsive (range 0.0 to 1.0)
        * **altitude\_mode** ([*FollowAltitudeMode*](#mavsdk.follow_me.Config.FollowAltitudeMode "mavsdk.follow_me.Config.FollowAltitudeMode")) – Follow Altitude control mode
        * **max\_tangential\_vel\_m\_s** (*float*) – [m/s] Maximum orbit tangential velocity relative to the target, in meters per second. Higher value = More aggressive follow angle tracking.
        * **follow\_angle\_deg** (*float*) – [deg] Follow Angle relative to the target. 0 equals following in front of the target’s direction. Angle increases in Clockwise direction, so following from right would be 90 degrees, from the left is -90 degrees, and so on.
